N5 Master Vocabulary List

Comprehensive list of essential Japanese vocabulary for JLPT N5 level.

About This List

This master list contains all the essential vocabulary for the JLPT N5 level, organized by categories. Each word includes the Japanese writing, romaji pronunciation, English meaning, and a category tag that links to the detailed category page.

Study Tip: The JLPT N5 level typically requires knowledge of around 800 vocabulary words. This list covers all those words and more. Use the search function (Ctrl+F) to quickly find specific words, or click on the category tags to see more context and examples for each word.
