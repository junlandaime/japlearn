Practice Exercises

Test your understanding of transitive and intransitive verbs with these practice exercises.

Exercise 1: Identify the Verb Type

Determine whether each verb is transitive (他動詞) or intransitive (自動詞).

1. 開ける (あける)

他動詞 (Transitive) - "to open something"

2. 始まる (はじまる)

自動詞 (Intransitive) - "to begin (by itself)"

3. 消す (けす)

他動詞 (Transitive) - "to turn off/extinguish something"

4. 壊れる (こわれる)

自動詞 (Intransitive) - "to break (by itself)"

5. 落とす (おとす)

他動詞 (Transitive) - "to drop something"

Exercise 2: Choose the Correct Particle

Select whether を (o) or が (ga) should be used in each sentence.

1. ドア＿＿開きました。

ドアが開きました。(Doa ga akimashita.)

The verb 開く (aku) is intransitive, so we use が (ga).

2. 先生は本＿＿閉めました。

先生は本を閉めました。(Sensei wa hon o shimemashita.)

The verb 閉める (shimeru) is transitive, so we use を (o).

3. 電気＿＿消えました。

電気が消えました。(Denki ga kiemashita.)

The verb 消える (kieru) is intransitive, so we use が (ga).

4. 私は窓＿＿開けました。

私は窓を開けました。(Watashi wa mado o akemashita.)

The verb 開ける (akeru) is transitive, so we use を (o).

5. コンピューター＿＿壊れました。

コンピューターが壊れました。(Konpyūtā ga kowaremashita.)

The verb 壊れる (kowareru) is intransitive, so we use が (ga).
